RxJava介绍

​ 一个在 Java VM 上使用可观测的序列来组成异步的、基于事件的程序的库。

​ 其优点是链式写法，代码简洁，可读性高。

观察者模式

​ 观察者模式是常用设计模式之一。a对象对b对象某一些变化高度敏感，需要一经改变就触发时，我们采用观察者模式。最常见的实现就是android中的OnClickListener()。我们通过setOnClickListener将监听传递给按钮，按钮在判断点击后，调用监听的方法。
